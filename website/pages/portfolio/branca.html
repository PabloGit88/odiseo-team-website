Branca Fotos

Aplicación Híbrida de fotos.

Permite a los usuarios realizar sesiones de fotos y agregar destinatarios para cada una de ellas. Luego, si los usuarios lanzan la sincronización, las fotos son almacenadas en la nuebe y enviadas por email a los destinatarios.

Tecnologías:

  • Cordova
  • Angular.js
  • Ionic
  • CSS3 & HTML5
  • Facebook Rest Api
  • PHP
  • MySql
  • Symfony
  • Jquery Libraries
  • HTML5
  • CSS3
  • Web Real-Time Communication (WebRTC) API
