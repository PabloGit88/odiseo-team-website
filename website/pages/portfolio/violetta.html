GUM - Violetta
Web para promocionar la marca mediante un concurso cuyos premios no sorteados se acumulaban día a día. La lógica para elegir el ganador se basaba en un algoritmo randómico que asignaba a los participantes una probabilidad de ganar de acuerdo a la cantidad de participantes y los premios disponibles hasta el momento.

Características:

  • Diseño responsable para desktop y móbiles.
  • Sistema para gestión de contenidos.
  • Integración con redes sociales.
